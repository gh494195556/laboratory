组件与Props

组件，从概念上类似于 JavaScript 函数。它接收任意入参(props)，返回一个React 元素。

直白点说就是，组件有点像函数，接收一个 props，返回一个 React 元素

函数组件与类组件

函数组件： 
function Welcome(props) {
  return <h1>Hello {props.name}</h1>;
}
类组件： 
class Welcome extends React.Component {
  render() {
    return <h1>Hello {props.name}</h1>;
  }
}

组件的渲染

一个 React 元素不仅仅可以包含 html 元素，也可以包含 其他 React 元素，例如： 

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}

const element = <Welcome name="Sara" />;

组件必须以 大写字 母开头，React 根据组件的开头字母的大小写区分是原生DOM标签或React组件，React组件上的 attributes 会在其 props 中体现。比如上面的代码，Welcome 的 attributes name 可以在 函数组件 Welcome 的 props 中使用。

提取组件

提取组件实际上和代码重构有点像，应该本着一个组件只对一个功能做描述的思想来提取代码。组件的内部应该尽可能的独立、可复用。我感觉组件的提取需要一定的时间去思考，可能每个人到最后的理解都会有所不同。

保持 props 只读

react 强烈建议不直接修改 props。
