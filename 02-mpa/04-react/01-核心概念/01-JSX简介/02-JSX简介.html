JSX简介

const element = <h1>Hello World!</h1>

这就是 JSX 语法，他既不是 string 也不是 html。

JSX 编译后会转换成普通的 JavaScript 代码，得到一个 JavaScript 对象。

在 React 中使用 JSX 时，代码最终会被 Babel 解析成 React.createElement()。

JSX 有以下这下特点：

  • 可以当成对象使用
  • 内部支持表达式
  • class 需要写成 className
  • tabindex 需要写成 tabIndex
  • 防注入
