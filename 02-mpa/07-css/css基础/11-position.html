元素定位方式有4种

  • Static
  • Fixed
  • Relative
  • Absolute

只有当设置过 position 属性后，top/left/bottom/right 才正常工作

Static

HTML元素的默认值，表示没有定位，出现在正常文档流，此定位 top/left/bottom/right 不可用

Fixed

相对浏览器窗口定位，不会受到滚动的影响，与文档流无关，会产生元素重叠

Relative

Relative 相对于 Static 定位

Absolute

Absolute 相对于最近的一个 已定位 的父元素，如果所有父级元素都没有定位，则相对 <html> 元素定位

Absolute 定位后不占据文档流的位置

因此，会产生重叠





z-index

z-index 可以设置重叠元素的层级，值越大，层级越高，越优先显示。

absolute.clip

absolute定位的元素可以 clip
