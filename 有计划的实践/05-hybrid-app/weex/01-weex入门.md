# Weex 快速入门

## 搭建开发环境

**1. 安装 weex-toolkit**

```
npm i -g weex-toolkit
```

**2. 检查开发环境**

```
weex doctor
```

## 创建一个 weex 项目

```
weex create weex-app
```

## 下载 weex playground

[weex playground](https://weex.apache.org/zh/tools/playground.html)

打开 weex playground 扫一扫即可查看效果
